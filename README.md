# 飞书自动记账

基于飞书群聊的自动记账工具，通过解析群聊消息自动提取记账记录并导出为 CSV 文件。

## 功能特性

- ✅ 自动获取飞书群聊历史消息
- ✅ 智能解析记账消息（格式：`整数金额 大类 具体原因`）
- ✅ 时间戳过滤，仅统计新增记录
- ✅ 自动计算总消费金额
- ✅ 发送统计结果到群聊
- ✅ 导出到 CSV 文件（全局记录 + 分时段记录）

## 配置说明

在 `config/config.yaml` 中配置飞书应用信息：

```yaml
env:
  APP_ID: cli_xxxxxxxxxxxxx
  APP_SECRET: xxxxxxxxxxxxxxxxxxxx
  chat_id: oc_xxxxxxxxxxxxxxxxxxxx
```

### 获取配置信息

1. **APP_ID / APP_SECRET**
   - 访问 [飞书开放平台](https://open.feishu.cn/)
   - 创建应用并获取凭证

2. **chat_id**
   - 在飞书群聊设置中查看群 ID
   - 或通过 API 获取群聊列表

## 使用方法

### 1. 安装依赖

```bash
uv sync
```

### 2. 发送记账消息

在飞书群聊中发送记账消息，格式为：

```
整数金额 大类 具体原因
```

示例：
```
20 吃饭 陈香贵牛肉面
15 零食 可乐
100 工作打车
```

### 3. 运行程序

```bash
python main.py
```

### 4. 查看结果

程序会自动：
- 将记录追加到 `result/global.csv`
- 创建时间戳文件夹（如 `20260330_131639/`）
- 在文件夹中保存当次的 `result.csv`
- 发送成功消息和总金额到群聊

## 记账格式

### 基本格式

```
金额 大类 具体原因
```

### 格式要求

- **金额**：必须是整数（如 `20`，不能是 `20.5`）
- **大类**：按空格分隔（如 `吃饭`、`零食`、`娱乐`、`工作`、`其他`）
- **具体原因**：可选，用于详细说明（如 `陈香贵牛肉面`）

### 示例

```
20 吃饭 陈香贵牛肉面
15 零食 可乐
100 工作 打车去客户
50 娱乐 电影票
```
